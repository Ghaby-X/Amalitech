# Scripting and Cloud Fundamentals

Labs covering shell scripting, configuration file formats, and AWS CLI
automation. Each subdirectory is a self-contained exercise with its own
README.

## Directories

| Directory | Purpose |
|---|---|
| `yaml_exercise/` | Constructs a YAML application configuration, validates it, and converts it to JSON with `yq`. |
| `aws_resource_creation/` | Bash scripts that provision (and tear down) an EC2 instance, security group, and S3 bucket via the AWS CLI. |
