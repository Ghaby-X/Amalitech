# Applied AI and Prompt Engineering

Labs on designing prompts and AI-assisted workflows. Each subdirectory is
a self-contained exercise with its own README.

## Directories

| Directory | Purpose |
|---|---|
| `multistage-ai-workflow/` | Chains a chat-based AI (drafts a restaurant landing page spec) into a CLI-based AI (implements it as a working static site), demonstrating prompt chaining across AI UX types. |
| `ai-assisted-technical-documentation/` | Generates Getting Started, API Reference, and Troubleshooting docs for a fictional app, then fact-checks and refines them against a source-of-truth spec through iterative prompting. |
| `ai-driven-data-insights-report/` | Analyzes a real UrbanTransit ridership dataset with AI assistance, fact-checking every claim against computed statistics and charts before writing a plain-English insights report with recommendations. |
