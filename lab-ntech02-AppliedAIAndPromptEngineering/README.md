# Applied AI and Prompt Engineering

Labs on designing prompts and AI-assisted workflows. Each subdirectory is
a self-contained exercise with its own README.

## Directories

| Directory | Purpose |
|---|---|
| `multistage-ai-workflow/` | Chains a chat-based AI (drafts a restaurant landing page spec) into a CLI-based AI (implements it as a working static site), demonstrating prompt chaining across AI UX types. |
