### Sprint 1

**Goal**: Deliver a working quiz flow where a user can start a quiz, answer multiple choice and true/false questions, and receive instant feedback on their answers.

**Total story points**: 7 (SP-1 + SP-3 + SP-3)

**User Stories**

| # | Story | Points |
|---|-------|--------|
| 1 | As a user, I want to start a quiz with one tap so I can begin answering questions quickly | SP-1 |
| 2 | As a user, I want to be able to answer either multiple choice or True/False questions so that I can test my knowledge | SP-3 |
| 3 | As a user, I want to be able to get instant feedback so that I can know whether my answers are correct or not | SP-3 |

**Sprint Tasks**

- [ ] Scaffold React + Vite project in `agile-devops-lab/web`
- [ ] Create question data (multiple choice + true/false)
- [ ] Build Home page with Quick Start button
- [ ] Build Quiz component with question rendering (MC + T/F)
- [ ] Implement single-selection answer logic
- [ ] Implement Submit button and answer evaluation
- [ ] Implement instant feedback (green correct, red wrong)
- [ ] Write unit tests for answer evaluation logic
- [ ] Set up GitHub Actions CI pipeline (build + test)
- [ ] Verify production build passes
